# FedGraph
Альтернативный экзамен по КиТГ 2022. Тема: "Федеративные вычисления на графах"

<h3>Поставнока задачи</h3>

На каждом из трех компьютеров( в локальной одноранговой сети, компьютеры могут быть как реальными так и виртуальными машинами)  в конфигурационном файле хранится информация о частях единого графа. Пользователь может ввести номера двух вершин графа с любого из устройств в сети, а  система произведет  поиск кратчайшего пути между этими вершинами.
